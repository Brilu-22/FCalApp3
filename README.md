## Technology Specifications

### Frontend Stack
- **Framework**: React Native 0.81.5 with Expo 54
- **Language**: TypeScript 5.9 with strict typing
- **Navigation**: Expo Router 6.0 (File-based routing)
- **State Management**: React Context + Async Storage
- **UI Components**: Custom design system + Ionicons
- **Styling**: StyleSheet with design tokens

### Backend Stack
- **Framework**: ASP.NET Core 8.0
- **Database**: Entity Framework with SQL Server
- **APIs**: RESTful design with JSON serialization
- **Authentication**: JWT tokens with Firebase integration


### AI & Services
- **AI Engine**: Google Gemini 2.5 Pro
- **Nutrition**: Edamam Food Database API
- **Authentication**: Firebase Auth
- **Analytics**: Custom tracking system
